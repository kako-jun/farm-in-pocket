# Farm in Pocket - プロジェクト構想

最終更新: 2025-11-17

## プロジェクト名

**Farm in Pocket (FarmInPocket)**

「ポケットの中の農業」を意味し、個人向け・小規模感を強調。

### ブランディング

- **正式名称**: Farm in Pocket
- **リポジトリ名**: `farm-in-pocket`（短縮形で扱いやすい）
- **サイトタイトル**: Farm in Pocket
- **キャッチコピー**: ポケットの中の農業 - 零細農家・個人農家のための完全無料IoT農業システム

---

## プロジェクトの背景

### 解決したい課題

1. **高額なメーカー純正システム**
   - 農業用IoTシステムは初期費用30万円〜、年間保守費5万円〜が一般的
   - 零細農家・個人農家には導入の障壁が高い

2. **ラストワンマイルの課題**
   - ハウス内の温度管理
   - 水やり作業の自動化
   - 害獣対策
   - 成長記録（写真記録）

3. **設定の複雑さ**
   - 既存のIoTシステムは設定が複雑で専門知識が必要
   - 導入・運用に時間とコストがかかる

4. **長期運用コストの不透明性**
   - 10年間の総コストが見えにくい
   - ランニングコストが予測しづらい

### ターゲットユーザー

- **零細農家・個人農家**
  - 小規模なビニールハウスを運営
  - IT知識は限定的
  - 予算が限られている

- **趣味で農業をする人**
  - 家庭菜園の延長
  - 週末農業

- **農業スタートアップ**
  - 低コストでIoTを導入したい
  - プロトタイピング用途

## プロジェクトの目的

### 基本方針

1. **完全無料・オープンソース**
   - すべてのソフトウェアとドキュメントを無償提供
   - 利用料金やライセンス費用が一切不要
   - MITライセンスで公開

2. **ノーコンフィグ（設定不要）**
   - SDカードに書き込むだけで起動
   - 初期設定を最小限に
   - デフォルト値で即座に使える

3. **プラグイン方式**
   - 必要な機能だけを選択可能
   - モジュールの追加・削除が簡単
   - 機能拡張が容易

4. **長期運用を前提とした設計**
   - 10年間の運用コストを明示
   - 保守・運用の負担を最小化
   - ハードウェアの入手性を重視

## 提供する機能

### コアシステム

1. **Yoctoベースのカスタムイメージ**
   - Raspberry Pi向けに最適化
   - 軽量で高速
   - セキュアな設計

2. **Web監視ダッシュボード**
   - ブラウザベースの管理UI
   - モジュールの稼働状況を一覧表示
   - 簡易ログ確認機能
   - OTAアップデート対応

3. **Dockerベースのモジュール管理**
   - 各機能はコンテナ化
   - manifest.jsonで統一的に管理
   - 簡単なインストール・アンインストール

### 機能モジュール

| モジュール名 | 機能 | 対象ハードウェア |
|------------|------|----------------|
| `farminpocket-temp` | 温度・湿度監視 | DHT22, DHT11, DS18B20 |
| `farminpocket-irrigation` | 自動潅水制御 | 土壌湿度センサー、電磁弁 |
| `farminpocket-photo` | タイムラプス写真記録 | Raspberry Pi Camera, USB Webカメラ |
| `farminpocket-scare` | 害獣監視＋威嚇（光・音） | PIRセンサー、LED、スピーカー |
| `farminpocket-connect` | データ送信（LoRaWAN/Wi-Fi/MQTT） | LoRaWANモジュール、Wi-Fi |

### サポートツール

1. **SDカード書き込みツール**
   - GUI版（Electron/Tauri）
   - CLI版（Python）

2. **コスト試算ツール**
   - 初期費用・年間維持費・10年試算
   - Webベースの計算機

3. **事例報告テンプレート**
   - 導入事例を共有するためのフォーマット
   - コスト情報の標準化
